开坑React

发表于 2020-04-20

4月14日腾讯实习offer call + 邮件了，应该是稳了的，然后部门用的是React，我之前用的较多的是Vue，所以想提前适应一下。
我打算根据袁老师的课 + 官方文档进行适应学习（这里吹一波袁老师的课真的讲的很深），并根据自己的理解归纳总结。

React简介

React是一个库

在官方文档也可以看得到，React自称是一个“用于构建用户界面的JavaScript库”。我自己的理解就是，不像Vue官方语法比较多，Vue是框架，侵入性较强；而React就像在写原生JS，更多的功能交给第三方插件来完成。
React由Facebook开发维护，Angular应该是由谷歌开发维护的。

React的特点

  • 轻量，源代码很少
  • 原生，基于原生JS书写而成，依赖很少的库
  • 易扩展，对代码的封装程度低，所以React很多功能可以扩展
  • 组件化开发
  • 单向数据流
  • 用JS代码声明界面

在使用插件时，Vue用的比较多的都是Vue-Router、Vuex这种官方出的插件；而React用的比较多的都是第三方的强大的插件，可能需要自己去探索最佳实践。

Hello React

引入核心库 cdn

加上crossorigin，报错了会显示详细信息。
用核心库的 React.createElement 创建React元素对象，用react-dom库将react对象转化成真实的dom对象。
JSX语法被babel编译后，实质上也是 React.createElement。

创建元素

React.createElement(元素类型，元素属性，…children)

创建的是React元素，是虚拟DOM，本质是React对象。
参数1：元素类型，或者组件
参数2：元素属性
剩余参数：元素的子节点

JSX

js的扩展语法，需要使用babel进行转义。babel会将JSX语法编译成 React.createElement 的形式。
